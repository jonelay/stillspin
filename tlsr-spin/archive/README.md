# Archived Modules

These modules require Gyr-timescale simulations outside current scope.

- **kozai.py** — Kozai-Lidov secular dynamics for distant companion
- **gr_corrections.py** — General relativity precession via reboundx
- **period_analysis.py** — Zero-crossing period measurement

These were developed for modeling instability events but
the current simulation pipeline focuses on ~1000 year timescales where
these effects are negligible.
